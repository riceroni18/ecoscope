## Project Overview

EcoScope is an interactive biodiversity application designed to make exploring life on Earth more accessible through data visualization, species information, and geographic exploration.

The project combines biological datasets with modern software development to create a platform where users can browse species, explore taxonomic relationships, and better understand global biodiversity.

Beyond serving as a learning tool, EcoScope is also a personal software engineering project that allows me to strengthen my skills in Python, data management, application development, and scientific computing while applying them to a field I'm passionate about.

---

## Motivation

As someone with a background in biology and clinical genomics, I've always enjoyed making complex scientific information easier to understand.

My experience teaching environmental science at Newport Bay Conservancy showed me how curiosity grows when people can actively explore scientific concepts instead of simply reading about them. EcoScope is an extension of that idea—using software to make biodiversity data more approachable and engaging.

The project also provides an opportunity to build practical experience designing larger software systems while working with real biological datasets.
